# Data Science Portfolio — 4 Job-Ready Projects

Each project includes:
- `data/make_dataset.py` (reproducible realistic synthetic dataset)
- `notebooks/` (EDA → modeling → evaluation → business recommendations)
- `src/` (deployment-style scripts)
- `models/` artifacts created by notebooks/scripts
- `reports/` exported metrics/plots (created when you run notebooks/scripts)

## Projects
1. Customer Churn Prediction (Subscription SaaS)
2. Retail Demand Forecasting (Store × SKU)
3. Payment Fraud Detection (E-commerce)
4. NLP Support Ticket Auto-Routing

## Run
```bash
pip install -r requirements.txt
python run_all.py
```
